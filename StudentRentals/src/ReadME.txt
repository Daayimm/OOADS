Requirements:
    FR1. Users must be able to register as Student or Homeowner.
    FR2. Homeowners must be able to create a room listing with:
        	rental_id,homeowner_id,student_id,rental_type,price,status,city,address
    FR3. Students must be able to filter listings by:
        Max Price , Rental Type , City
    FR4. Students must be able to request a booking
